# 轮播图助手

微信公众号图文编辑页面的图片采集与轮播代码生成工具。

## 功能介绍

### 核心功能

- **一键采集** — 自动抓取编辑区内所有已上传图片的真实 URL
- **点击采集** — 点击编辑区任意图片直接采集，无需切换工具
- **图片管理** — 上移、下移调整顺序；删除、清空；手动添加链接；自动去重
- **轮播代码生成** — 根据采集图片数量自动生成可横向滑动的轮播 HTML
- **尺寸统一** — 选择参考图片高度，自动统一所有图片尺寸
- **两种复制模式** — 复制纯文本代码（可查看修改）或复制富文本（直接粘贴显示轮播效果）

### 面板操作

- **竖向悬浮入口** — 右侧小按钮点击展开面板
- **拖拽移动** — 拖动标题栏可自由调整面板位置
- **插件开关** — 可启用/禁用插件功能
- **代码预览折叠** — 点击 `▶ 轮播代码预览` 展开/收起代码

## 安装方式

1. 下载本项目
2. 打开 Chrome，进入 `chrome://extensions/`
3. 开启右上角**开发者模式**
4. 点击**加载已解压的扩展程序**
5. 选择项目文件夹

## 使用方法

1. 打开微信公众号图文编辑页面（`https://mp.weixin.qq.com/`）
2. 点击右侧悬浮按钮展开面板
3. **方式一**：点击「一键采集全部图片」抓取编辑区所有图片
4. **方式二**：直接点击编辑区的任意图片进行单张采集
5. 在面板中使用 ↑↓ 调整图片顺序
6. 在「尺寸统一」区域选择参考图片并点击「应用选中尺寸」
7. 点击「复制纯文本」或「复制富文本」获取轮播代码
8. 将代码粘贴到公众号编辑器中即可

## 效果说明

生成的轮播图：
- 支持手机端手指左右滑动切换图片
- 图片下方有双语引导文字 `←左右滑动查看更多→ Slide for more photos`
- 图片下方有圆点指示器
- 图片宽度自适应内容区，在手机端自动等比缩放完整显示

## 注意事项

- 仅适配微信公众号后台图文编辑页面
- 富文本复制模式在部分浏览器可能降级为纯文本
- 轮播代码中的交互提示依赖微信编辑器的解析规则，可能在发布后略有差异
